# Microsoft 365 Agent Toolkit

## Overview

A Visual Studio Code extension that helps developers create, test, and deploy AI-powered agents for Microsoft 365.

Benefits:

- Simplified agent development with built-in templates and tools.
- Seamless integration with Microsoft 365 services.
- Streamlined deployment to Microsoft 365 environments.
- Allows development of:
  - Custom AI agents tailored to organizational needs.
  - Agents that leverage Microsoft Graph data.
  - Agents that integrate with Microsoft Teams and SharePoint.

Capabilities:

- Declarative agents or Custom Engine agents.
- Copilot Connectors: They act as integration bridges that allow Copilot to access and reason over data from external systems
